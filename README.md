# Big4

A private space for four friends — built to stay close and get wealthy together.

**The crew:**
- Jakom (Derro)
- Kwach (Selekta)
- Omin (Memo)
- Omwanda (Tobby)

## Features
- 🔐 Login for each of the four members, admin-assisted password reset
- 💬 Group chat — mentions, emojis, file attachments, online status
- 🎯 Savings goals — set a target amount, target date, and travel date; any member can contribute
- 🖼 Gallery — each member has their own photo section
- 📣 Updates — a Facebook-style feed with posts, likes, and comments
- 🛠 Admin panel (Jakom only) — view all savings activity, manage member passwords

## Status
Current version is a front-end prototype using browser local storage as a stand-in database. Next step is connecting it to Supabase for real shared accounts, live chat, and cloud photo storage across devices.

## Tech
- HTML / CSS / JS (single-file, `big4.html`)
- Supabase (planned: Auth, Realtime, Storage, Postgres)
- Deployed via GitHub Pages
